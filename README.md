# taller-html-Mariana-CarvajalRueda

## Descripción 
Esta es una aplicación web muy sencilla que permite buscar razas de perros y ver diferentes fotografías de la raza.
La aplicación consume datos desde la API pública Dog CEO API, obteniendo imágenes de la raza que el usuario ingrese en el buscador y mostrándolas en una galería.

## Cómo ejecutar
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo:
"index.html"
4. Escribir una raza de perro en el buscador (por ejemplo: husky, pug, beagle).
5. Presionar el botón Buscar para visualizar las imágenes.
